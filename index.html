What is it and what is Python for?

Python is a high-level programming language that is used to develop applications of all kinds. Unlike other languages such as Java or .NET, it is an interpreted language, that is, it is not necessary to compile it to run the applications written in Python, but rather they are executed directly by the computer using a program called an interpreter, so that it is not necessary to “translate” it into machine language.


Python is an easy language to read and write due to its high similarity with human language. In addition, it is an open source multiplatform language and, therefore, free, which allows you to develop software without limits. Over time, Python has been gaining popularity thanks to its simplicity and its extensive possibilities, especially in recent years, since it makes it easier to work with artificial intelligence, big data, machine learning and data science, among many other booming fields. and in great demand.

Python Module Operator

Returns the remainder of the division of the left operand by the right operand.
It is used to find the remainder of a division problem.

The modulo operator is considered an arithmetic operation just like the operators +,  -,  /,  *,  **,  //.

The basic syntax is:

a % b

In the example above, a is divided by b and the remainder is returned.

Let's look at an example with numbers:

7 % 2

The result of the above example is one.
Two out of seven (Three times) and one left over.

The following diagram shows a visual representation of 7 / 2 and 7 % 2 (The "R" stands for remainder):

7 / 2 = 3 R 1
7 % 2 = 1

The R is the remainder of the division of the problem.
It is also the answer to 7 % 2.

Here's another example:

3 % 4

This will result in three.
The four does not fit in the three, so the original three is still left over.

The following diagram shows what is happening:

3 / 4 = 0 R 3
3 % 4 = 3

The rest is three.


Remember, the modulus operator returns the remainder after performing the division.

Variables

* It would be good to give these data types a name to be able to recognize them.

For example:

Variable naming rules

* It is considered good practice to use lowercase names.

* Avoid using words with special meanings such as "list" o "str".

Python uses Dynamic monenclature

* It means that we can re-assign variables to other data types.

* This makes python quite flexible when assigning variables.

For example:

My_houses = 2

or what would be the same:

My_houses = ["house1","house2"]

This is correct in Python!!

Text Strings

* Strings are sequences of characters and use single or double quoted syntax.

For example:

'hello'
"hello"
Character:    "Text string with many characters"
Index:          0      1    2    3       4
Index reverse:  0     -4   -3   -2      -1

Data:

Both string and list indices in Python always start at 0 (Zero).

* Because strings are ordered sequences, means we can use Indexing and Slicing to grab sub-sections of a string.

  1. [+] Indexing notation: [], assigned after the string.
  2. [+] Indexed: Allows us to grab a single character from a string of text.
  3. [+] Slicing: Allows us to grab a sub-section of multiple characters (A slice of a chain).
  4. [+] Syntax: [start:stop:step] -> [0:4:2].
  5. [+] Start: It is a numerical index for the slice to start.
  6. [+] Stop: It is the number where we stop the Slice (End).
  7. [+] Step: It's the steps we take.

We print a Text String:

print("hello world") -->> In one line.

print("hello \nworld") -->> In two lines.

We print the Length of a Text String:

print(len("hello world"))

Result: 10 -->> Spaces and special characters are also counted (Rare).
Index and Slicing of a String

Indexed:

mystring = "Hello World"
print(mystring[2])

Result: l.

Indexed reverse:

mystring = "Hello World"
print(mystring[-1])

Result: o.

Slicing #1:

mystring = "Hello World"
print(mystring[0:4])

Result: hello.

Slicing #2:

mystring = "Hello World"
print(mystring[0:4:2])

Result: hl.

Slicing #3:

mystring = "Hello World"
print(mystring[0:7:2])

Result: hl o.

Slicing #4:

mystring = "Hello World"
print(mystring[3:8])

Result: o Wor.

This:

mystring = "Hello World"
print(mystring[::])

Result: Hello World.

Would be the same as this:

mystring = "Hello World"
print(mystring)

Result: Hello World.
String Properties and Methods

Text String Immutability:

name = "Mich"
name[0] = 'P'

Result: TypeError: 'str' object does not support item assignment.

Which means Strings are Immutable.
They are mutated and can NOT be changed.

To be able to edit a string we must concatenate it:

name = "Mich"
last_lyrics = name[1:]

Result: ich.

We concatenate #1:

name = "Mich"
last_lyrics = name[1:]
print('M' + last_lyrics)

Result: Mich.

We concatenate #2:

name = "Mich"
last_lyrics = name[1:]
print('P' + last_lyrics)

Result: Pich.

Example #1:

Example #2:
